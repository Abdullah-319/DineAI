# Project Setup

## Requirements
- Python **3.11**
- **Deepgram API Key**
- **Gemini API Key**

## Steps to Run

1. Open the `.env` file and add your **Deepgram** and **Gemini** API keys.  
2. Create a virtual environment and install dependencies from `requirements.txt`:  
   ```bash
   pip install -r requirements.txt
3. Run the app:
   ```bash
   python app.py
